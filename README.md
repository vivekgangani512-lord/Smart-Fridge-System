# Smart Fridge System

A smart refrigerator inventory management system featuring barcode-based product tracking, expiry alerts, and modern glassmorphism UI design.

## Features
- Food Inventory Management
- Expiry Monitoring
- Barcode Scanning Workflow
- Category-based Product Organization
- Modern Glassmorphism UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Barcode API
